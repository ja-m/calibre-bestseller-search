# Calibre Bestseller Search

Eine interaktive Webseite zur Suche und Anzeige von Bestseller-Büchern aus Calibre.

## 🌐 Live-Seite

👉 [ja-m.github.io/calibre-bestseller-search](https://ja-m.github.io/calibre-bestseller-search/)

## 📖 Beschreibung

Diese Seite ermöglicht die Suche und Filterung von Büchern aus einer Calibre-Bibliothek. Sie wurde als statische HTML-Seite umgesetzt und läuft direkt im Browser – ohne Backend.

## 🚀 Technologie

- HTML5 / CSS3 / JavaScript
- Gehostet via GitHub Pages
